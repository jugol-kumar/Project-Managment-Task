### Module: Project Management
## Features:
- Create project (title, client, start/end date, status).
- Add tasks to project (title, deadline, assigned_user, status).
- API endpoints for:
- - • Create Project
- - • Add Task to Project
- - • Retrieve Project with Tasks (optimized query) Implement Policies or Gates for permissions.
- - Implement queue job that sends email notification after task creation.

- Create a simple Vue.js or Blade front-end (optional but adds points).





Installation Process
- Clone git repository
- Install composer and node dependencies
- Generate tables form migrations
- Run server
- Explore this app
